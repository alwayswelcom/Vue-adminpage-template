# vueAdmin
a simple admin portal based on [vuejs2](http://vuejs.org/) and [element](http://element.eleme.io/#/).

### Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).

``` bash
$ npm install
$ npm run dev
$ npm run build
```
