# vueAdmin
基于 [vuejs](http://vuejs.org/) , [element](http://element.eleme.io/#/) 搭建一个后台管理界面

### Usage

这个项目是使用[vue-cli](https://github.com/vuejs/vue-cli)生成

``` bash
$ npm install
$ npm run dev
$ npm run build
```
